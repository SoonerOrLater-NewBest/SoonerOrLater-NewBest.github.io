# 静态博客

# 简介

本站就是演示

  • 技术栈：Vue + Vuepress + Element-Ui + MarkDown + Axios + NodeJs
  • 目前主要用 vuepress 进行编译，使用 MD 撰写静态文档，使用 vue+ele 自定义组件，拓展布局，使用 Axios 跨域请求一些配置类数据，由 nginx 反向代理 nodejs 应用，提供接口
  • 尝试加入 css 特效和 canvas 特效，一切为了学习（为了秀而秀，请使用谷歌浏览器查看）
