# Mac

  • rm -rf xx // 删除文件
  • sudo lsof -i :8080 // 查看指定端口是否被占用，返回PID
  • kill -9 PID // 并杀死进程
  • Mac解压命令

# LINUX

  • mysqladmin -uroot -p shutdown // 关闭mysql
  • mysqld_safe & // 启动mysql
  • chkconfig --list // 查看开机自动启动的服务使用chkconfig命令，其中mysql的2~5为on，说明mysql服务会随机器启动而自动启动
  • netstat -tulpn // 查询端口
  • ps -ef grep mysqld // 查询进程，如果有mysqld_safe和mysqld两个进程，说明MySQL服务当前在启动状态
  • GRANT ALL PRIVILEGES ON . TO '账号'@'%' IDENTIFIED BY '密码' WITH GRANT OPTION; //2、设置最高权限的mysql账号
  • Linux命令大全

# NPM

  • npm config set registry https://registry.npm.taobao.org // 设置淘宝源
  • npm config get registry // 查看源，可以看到设置过的所有的源
  • npm install -g cnpm --registry=https://registry.npm.taobao.org // 安装cnpm命令,不会改变npm的源
  • npm list -g --depth 0 // 查看本地全局安装过的npm包
  • eslint --fix file // ESlint自动修复
  • npm config get prefix // 查看npm全局模块路径
  • npm config delete prefix // 删除
  • npm config set prefix xxxxx //设置

# Git

Git共有三个级别的config文件，分别是system、global和local

  • git config --global -l // 查看全局配置
  • git config --local user.name LemonCat // 设置当前仓库用户名
  • git config --local user.email lemoncat@***.com // 设置当前仓库邮箱
  • git config --global credential.helper store // 缓存账号
