# 宜搭简介

宜搭是阿里巴巴自研的低代码应用构建平台，通过可视化拖拽的方式，传统模式下需要两周才能完成开发的应用，用宜搭 2 小时就能完成。 专业软件对于实际的工作而言，功能冗余，价格不菲，还要求助公司专业开发去做。宜搭让人人都可以成为开发者，通过拖拉拽就能搭建自己的应用，像积木拼搭一样，小的审批，大的门户系统，自己就能搞定。

对于不会代码的普通人而言，也不用求人，还能体验当开发者的乐趣。对于专业的开发者而言，用宜搭可以摆脱重复低效的工作模式，把精力集中在更有价值的事情上。宜搭还提供了适合各行各业的免费应用模板，开箱即用，一分钟就能打造一款专属应用。
