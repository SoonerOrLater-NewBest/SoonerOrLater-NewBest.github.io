# 使用vuepress

# 安装全局vuepress

npm install -g vuepress
1

COMPATIBILITY NOTE

VuePress requires Node.js >= 8.

# 运行程序

运行

npm run docs:dev
1

或者直接运行

vuepress build docs
1

# 网站配置

想要定制你的网站，首先需要在docs目录中创建一个 .vuepress 目录,这是放置所有 VuePress 特有(VuePress-specific) 文件的地方,然后再该文件下创建config.js文件




 









module.exports = {
title: 'Vue插件使用', //网站标题和左上角名称
port：8080, //网站端口
dest：'.vuepress/dist', //发布文件路径配置
description: 'Vue 插件介绍', //网站描述
themeConfig: { //网站主题配置，可以看看官网 https://vuepress.docschina.org/
  sidebar: [  //左侧菜单栏，也可以通过配置nav来配置网站右上角导航
    ['/', '网站介绍'], //默认指向docs根目录下的READEME.md文件
    ['/api/', 'vuepress介绍'], //默认指向docs/api/目录下的READEME.md文件
  ],
}
}

# 发布网站

发布命令

npm run docs:build
1

或者直接使用

vuepress build docs
1

发布成功提示如下，默认会在docs/.vuepress目录下生成dist文件所有静态文件都在该文件下（可以在config.js文件下配置dest项控制发布文件路径），直接发布该文件到服务器即可
