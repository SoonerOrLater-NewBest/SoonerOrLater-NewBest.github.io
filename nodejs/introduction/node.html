# 原理

1.1 Node.js 简介 Node.js 是基于事件驱动的单进程单线程应用，单线程具体体现在 Node.js 在单个线程中 维护了一系列任务，然后在事件循环中不断消费任务队列中的节点，又不断产生新的任 务，在任务的产生和消费中不断驱动着 Node.js 的执行。从另外一个角度来说，Node.js 又可以说是多线程的，因为 Node.js 底层也维护了一个线程池，该线程池主要用于处理一 些文件 IO、DNS、CPU 计算等任务。
